== Description ==
Bindet einen kleinen Währungs-Umrechner der Webseite [https://www.curvert.com](https://www.curvert.com?utm_source=wordpress-plugin) in deine Seite als Sidebar-Widget ein.


== Installation ==
1. Lade das Plugin in das `/wp-content/plugins/` Verzeichnis 
2. Aktiviere das Plugin über das 'Plugin'-Menü in deinem WordPress
